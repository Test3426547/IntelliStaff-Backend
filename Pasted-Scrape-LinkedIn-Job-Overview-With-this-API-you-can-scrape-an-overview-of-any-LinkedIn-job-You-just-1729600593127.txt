Scrape LinkedIn Job Overview
With this API you can scrape an overview of any LinkedIn job. You just have to pass the ID of that job.

You have to send a GET request to 
http://api.scrapingdog.com/linkedinjobs
 with the below-given parameters.

Parameters
Parameter	Description
api_key

required

Your personal API key. Available on your dashboard

Type: String

job_id

required

This is the ID of the job. This can be found through our LinkedIn Jobs Scraper or from the LinkedIn Job URL.

API Example
cURL
Python
Nodejs
PHP
Ruby
Java
Copy
const axios = require('axios');

const url = 'https://api.scrapingdog.com/linkedinjobs';
const api_key = '5eaa61a6e562fc52fe763tr516e4653';
const job_id = '3539667535';

const params = {
  api_key,
  job_id,
};

axios
  .get(url, { params })
  .then((response) => {
    if (response.status === 200) {
      console.log(response.data);
    } else {
      console.error(`Request failed with status code: ${response.status}`);
    }
  })
  .catch((error) => {
    console.error(`Request failed: ${error.message}`);
  });
